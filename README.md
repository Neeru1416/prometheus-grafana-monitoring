# prometheus-grafana-monitoring# Prometheus Grafana Monitoring Stack

## Overview

Monitoring stack built using Prometheus and Grafana for infrastructure and application monitoring.

## Components

- Prometheus
- Grafana
- Node Exporter
- MySQL Exporter
- Blackbox Exporter

## Features

- Infrastructure Monitoring
- Database Monitoring
- URL Health Monitoring
- Dashboard Visualization
- Alerting Support
